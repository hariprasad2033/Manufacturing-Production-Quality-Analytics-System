# Manufacturing Production & Quality Analytics System

## Project Overview
This project is an end-to-end Manufacturing ETL and Analytics solution built using Informatica IICS, Oracle Database, and Power BI.

## Features
- Dynamic CSV ingestion
- Data validation and reject handling
- Lookup transformations
- Router transformations
- Aggregator and Rank analytics
- Automated alert generation
- Taskflow orchestration
- Power BI dashboards

## Technologies Used
- Informatica IICS
- Oracle Database 11g
- Power BI
- SQL
- CSV Flat Files

## Workflow
CSV Files → Informatica IICS → Oracle Database → Power BI Dashboard
